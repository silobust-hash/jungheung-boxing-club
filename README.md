## 기술 스택

- HTML + Tailwind CSS (CDN)
- Vanilla JavaScript
- 단일 파일 정적 사이트

## 로컬 실행

```bash
# Python 정적 서버
python -m http.server 8080

# 또는 npx serve
npx serve .
```

브라우저에서 `http://localhost:8080` 접속.
